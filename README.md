# Retail Sales Management System

## Overview
A MySQL database project for managing customers, products, and sales transactions while generating business analytics reports.

## Features
- Customer Management
- Product Management
- Sales Tracking
- Revenue Analysis
- Views
- Stored Procedures
- Triggers

## Database Schema

Tables:
- customers
- products
- sales
- sales_items
- sales_audit

## Views
- customer_sales_summary
- monthly_sales_summary
- product_sales_summary

## Technologies Used
- MySQL 8.0
- MySQL Workbench

## Sample Reports
- Top Customers
- Monthly Revenue
- Product Performance
## How to Run
1. Open MySQL Workbench
2. Execute retail_sales_management.sql
3. Run report queries
